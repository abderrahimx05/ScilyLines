# SicilyLines
# La compagnie SIclly Lines est une compagnie maritime privée, dans le domaine du transport rapide de passagers avec leurs véhicules entre plusieurs villes. SIclly veut améliorer son service de réservation en ligne,donc les informations seront générées à partir d’une base de données ,qui doit contenir les liaisons entre les secteurs, avec un tarif qui dépend  de la catégorie des passagers , les horaires de départ et d’arrivée
